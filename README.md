## Зависимости
```
// Библиотека gRPC
$ go get google.golang.org/grpc

// Загрузка протобафа со схемами gRPC
$ go get github.com/DoomerAlex/gRpcAuthProtos

// Библиотека для чтения конфигов в yaml формате 
$ go get github.com/ilyakaznacheev/cleanenv
```

## Структура проекта:
```
| - cmd ....................Команды для запуска приложения
|    | - migrator ..........Утилита для миграций баз данных
|    | - sso ...............Основная точка входа в сервис
| - config .................Конфигурационные файлы
| - internal ...............Внутренности проекта
|    | - app ...............Код для запуска различных компонентов приложения
|    |    | - grpc .........Запуск gRPC-сервера
|    | - config ............Загрузка конфигурации
|    | - domain ............
|    |    | - models .......Структуры данных и модели домена
|    | - grpc...............
|    |    | - auth..........gRPC-хэндлеры сервиса Auth
|    | - lib................Общие вспомогательные утилиты и функции
|    | - services...........Сервисный слой (бизнес-логика)
|    |    | - auth..........Компонениты авторизации
|    |    | - permissions...Компоненты аутентификации
|    | - storage............Слой хранения данных
|         | - sqlite........Реализация на SQLite
| - migrations..............Миграции для БД
| - storage.................Файлы хранилища
| - tests...................Функциональные тесты
```
